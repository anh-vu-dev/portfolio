# DevOps Portfolio Landing Page

A simple personal portfolio website built to showcase my background as a Backend Engineer and my hands-on DevOps learning through real deployment practice.

## Overview

This project is a static HTML landing page that introduces my profile, highlights my DevOps experience, and links to live deployment environments and source code.

It was created as part of my portfolio-building process to support job applications and demonstrate practical execution beyond a traditional CV.

## Features

- Personal introduction
- Backend and DevOps profile summary
- Project showcase with live demo links
- GitHub profile link
- Multi-platform deployment visibility
